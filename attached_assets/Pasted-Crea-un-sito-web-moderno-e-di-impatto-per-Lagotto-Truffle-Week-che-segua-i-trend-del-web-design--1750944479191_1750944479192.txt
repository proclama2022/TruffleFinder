Crea un sito web moderno e di impatto per "Lagotto & Truffle Week" che segua i trend del web design 2025, mantenendo le grafiche originali del documento ma elevandole con effetti eleganti e gestibili.
DESIGN VISION - MODERNO ED ELEGANTE:
HOMEPAGE HERO:

Hero section full-height con gradient background animato (#8B4513 to #DAA520)
Logo circolare centrale con subtle rotation al hover
Typography bold e moderna con fade-in animation
CTA button con gradient e hover glow effect
Scroll indicator elegante con bounce animation
Background con subtle pattern di foglie autunnali

DESIGN SYSTEM MODERNO:
Palette Colori:

Primario: #8B4513 (marrone tartufo)
Secondario: #DAA520 (oro elegante)
Gradient: Linear da marrone a oro per elementi hero
Neutri: #FAF0E6, #F5F5DC per backgrounds
Dark mode: Toggle con smooth transition

Layout Moderno:

Cards con glassmorphism: Background semi-trasparente con blur
Shadows moderne: Box-shadow multi-layer per depth
Border radius: 16px per look contemporaneo
Spacing system: Grid 8px per consistency
Typography scale: Hierarchy chiara e leggibile

SEZIONI PRINCIPALI:
1. HERO SECTION:
- Background gradient con overlay pattern
- Logo animato con CSS transforms
- Titolo con typing animation semplice
- CTA button con hover scale effect
- Stats counter animato (giorni, attività, partecipanti)
2. PROGRAMMA - TIMELINE ELEGANTE:
- Timeline verticale su desktop, cards su mobile
- Ogni giorno ha una card con hover lift effect
- Icone SVG per ogni attività
- Progress line che si riempie allo scroll
- Date prominenti con typography bold
3. ATTIVITÀ - GRID MODERNO:
- CSS Grid responsive con gap generoso
- Cards con immagine, gradient overlay e testo
- Hover effects con smooth transitions
- Badge per difficoltà/durata
- Bottoni ghost style
4. GALLERY - MASONRY RESPONSIVE:
- Layout a griglia asimmetrica
- Lightbox semplice per zoom
- Lazy loading per performance
- Filtri con active states
- Hover overlay con info
